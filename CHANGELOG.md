# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-03

### Added
- Initial public release
- Complete Schema-First prototyping template
- Three AI contract-maintenance skills:
  - `schema-evolution` - Detects changes and proposes schema updates
  - `concept-sync` - Keeps documentation synchronized
  - `contract-validator` - Validates consistency across artifacts
- Working ticketing system example with:
  - Complete OpenAPI 3.1 schema
  - Domain and behavior models
  - Generated TypeScript types
  - Mock API implementation
  - React UI prototype
- Comprehensive documentation:
  - Workflow walkthrough
  - Schema-First vs Traditional comparison
  - Skills writing guide
  - Agent reference (AGENTS.md)
- React 19 + TypeScript + Vite UI template
- Sailwind component library integration
- Aurora color palette design system
- MIT License
- Contributing guidelines
- GitHub issue and PR templates

### Documentation
- README.md with quick start guide
- CONTRIBUTING.md with contribution guidelines
- LICENSE file (MIT)
- Example-specific README for ticketing system
- Complete docs/ folder with guides

### Infrastructure
- Clean project structure separating template from examples
- TypeScript configuration
- ESLint configuration
- Vite build setup
- Git ignore patterns
